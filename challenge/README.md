# Terraform SSM Challenge

This chart is responsible to deploy SSM parameters in AWS cloud.

#### Dependencies

- AWS CLI
- Terraform
- [asdf](https://asdf-vm.com/) (optional)

#### Region

- sa-east-1

#### Init

```bash
make init
```

#### Plan

```bash
make plan
```

#### Apply

```bash
make apply
```

#### Destroy

```bash
make destroy
```
